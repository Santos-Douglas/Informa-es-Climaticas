Siglas:
para comentar usamos /*  */

<head> - Onde está os stilos da aplicação (cabeça).
<H2> - Titulo
<p> - Texto
stylesheet - folha de estilo em cascata
* - Significa tudo - você vai selecionar todos os elementos do HTML
margin - serva para distanciar a parte de fora do componente, quadro.(externo)
padding - é um espaçamento dentro do componente, quadro. (interno)
box sizing - 
width - largura
display - ele determina como os elementos aparece na tela.
height vh- altura
width vw - largura
font size - tamanho do texto
place holder - é o tempo que fica na barra de digitação
outline - linha quando selecionado o elemento.
border radius - arredondamento das bordas
opacity - transparencia do item
float - jogar para o lado que quiser (right\left)
button:hover - é o evento de passar o mouse por algum lugar.
cursor - para quando o mouse passar em uma região aparece uma mão.
async - serve para acionarmos ou acessar o servidor.
await - para buscar as informações ele demora alguns segundos e ele
serve para aguardar até as info chegar.
fetch - é a ferramenta que vai até o servidor e puxar as info
innerHTML - sempre quando você quer inserir algo do html
Math.floor - é responsável por fazer o arredondamento das info no caso os (graus °C)
text transfor - capitalize - deixar sempre a 1ª letra maiuscula.
